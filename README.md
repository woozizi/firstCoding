영화 검색 사이트

프로젝트 소개

TMDB의 API를 활용하여 인기영화를 확인할 수 있고 원하는 영화를 검색할 수 있습니다.
HTML, CSS JavaScript를 활용한 프로젝트 입니다.

주요기능
- 인기 영화를 확인 가능
- 영화별 상세정보 확인 가능
- 검색을 통해 원하는 영화 정보 확인 가능
- 북마크를 추가해 내가 보고싶은 영화목록을 따로 확인 가능
- 실시간으로 화면이 동기화 가능

기술 스택
HTML
CSS
JavaScript

프로젝트 구조
project-directory/
├── index.html      # 메인 HTML 파일
├── styles.css      # 전체 스타일링
├── modal.css       # 모달 창 스타일링
├── app.js          # 주요 로직 및 기능
├── ui.js           # UI 제어 로직
└── api.js          # API 요청 및 데이터 처리
